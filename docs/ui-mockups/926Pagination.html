Component Variations

Variant Name : "Standard Pagination"

The default, full pagination control for desktop and tablet screens, showing page numbers.

Variant Name : "Simple Pagination (Responsive)"

A compact version for mobile screens, showing only Previous and Next buttons. This is not a class-based variant, but the responsive state of the standard component.
