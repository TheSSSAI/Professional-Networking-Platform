UserMenu Component

Provides users with a consistent entry point to access their profile, settings, and logout functionality. The component combines an Avatar with a dropdown menu.

Default (Closed) State

This is the default appearance of the UserMenu, typically seen in the application header. The dropdown is hidden.

Open State

When the user clicks the toggle button, the dropdown menu appears. This state is simulated by adding the .is-open class to the main .user-menu container.
