Moderation Action Menu

A molecular component for displaying a set of contextual administrative actions.

Closed State (Default)

This is the default appearance of the component before user interaction.

Open State

This shows the menu when it's open, triggered by a click. Note the use of `.mod-action-menu--open`.

Live Demo (CSS-only)

Click the icon to open and close the menu. This demo uses a hidden checkbox for state management and does not represent the final JavaScript implementation.

Component Variations

Default State (Closed)

The standard, at-rest state of the component. The menu is hidden and the trigger is interactive.

Open State

The active state when the user has clicked the trigger. The menu is visible and its items are interactive.
