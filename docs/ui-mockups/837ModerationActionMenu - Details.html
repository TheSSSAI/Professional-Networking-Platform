Description

A context menu component that provides administrators with a set of moderation actions for a specific item, such as a reported post or a user account. It's typically triggered by an icon button.

Design System Alignment

Uses design system tokens for colors (primary, danger, neutral), spacing, border radius, and focus states. The component is composed of atomic Button and DropdownMenu patterns.

Accessibility Features

Keyboard Support

The trigger button is focusable and activatable with Enter/Space. Menu items are focusable with Tab and activatable with Enter/Space. The menu can be closed by pressing Escape (requires JS).

Screen Reader Support

The trigger button uses `aria-haspopup='true'` and `aria-expanded` to indicate its function and state. The dropdown has `role='menu'` and items have `role='menuitem'`.

Focus Management

Visible focus indicators are applied to the trigger and menu items. For a production component, JavaScript would be needed to trap focus within the open menu and return it to the trigger on close.

Aria Attributes

  • aria-haspopup
  • aria-expanded
  • aria-label
  • role='menu'
  • role='menuitem'

Color Contrast

All text and interactive elements meet WCAG 2.1 AA contrast requirements.
