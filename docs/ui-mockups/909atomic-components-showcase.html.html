Accordion

For collapsing and expanding sections of content.

What is an accordion?

An accordion is a vertically stacked list of items. Each item can be expanded or collapsed to reveal the content associated with that item.

Is this accessible?

Yes, using the native <details> and <summary> elements provides excellent accessibility out of the box, including keyboard navigation and screen reader support.
