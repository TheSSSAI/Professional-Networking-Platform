Description

A card component that displays a rich preview of an external URL, typically including an image, title, description, and source. It includes a loading state and is designed to be used within larger components like the PostCard or PostComposer.

Design System Alignment

This component is built upon the atomic 'Card' and 'SkeletonLoader' components. It uses design system tokens for spacing, typography, colors, border-radius, and shadows to ensure visual consistency with the rest of the application.

Responsive Specifications

Mobile Adaptations

  • The component is fluid and takes up 100% of its container's width.
  • Typography and spacing remain consistent with the mobile-first design system.

Tablet Adaptations

  • No specific adaptations needed; component scales with its container.

Desktop Enhancements

  • Hover effects are more pronounced and expected.
  • Higher resolution images should be served if an image optimization service is used.

Accessibility Features

Keyboard Support

The entire card is a single focusable link. The remove button is focusable and can be activated with Enter/Space.

Screen Reader Support

The main component is a semantic `` tag. The remove button has an `aria-label` for context.

Focus Management

Standard browser focus indicators apply to the link and the remove button.

Aria Attributes

  • `aria-label` on the remove button.

Color Contrast

Text colors on backgrounds meet WCAG 2.1 AA contrast requirements.
