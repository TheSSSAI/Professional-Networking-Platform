Description

A versatile file upload component that provides a drag-and-drop zone and visual feedback for idle, uploading, success, and error states. It is designed primarily for image uploads but can be adapted for other file types.

Design System Alignment

This component leverages atomic elements from the design system, including Button, LoadingSpinner, and Icon. It uses design system tokens for colors, spacing, typography, and borders to ensure visual consistency with the rest of the application.
