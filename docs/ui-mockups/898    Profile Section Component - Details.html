Responsive Specifications

Desktop Layout

  • Section is contained within a max-width card.
  • List items display full details with horizontal space for actions.

Tablet Adaptations

  • Layout remains largely the same, with reduced padding and margins to fit the viewport.

Mobile Adaptations

  • Card padding is further reduced.
  • Font sizes are scaled down slightly for readability.
  • Complex list items may stack elements vertically if needed to prevent crowding.
