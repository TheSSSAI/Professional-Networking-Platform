Responsive Specifications

Desktop Layout

  • Full-width table within a max-width content area.
  • Header contains title and primary actions.
  • Footer contains pagination controls.

Tablet Adaptations

  • Layout remains similar to desktop.
  • Table wrapper enables horizontal scrolling if the content overflows the viewport width.

Mobile Adaptations

  • Header elements may stack to conserve horizontal space.
  • Table is horizontally scrollable by default to prevent content from being squashed or truncated.
  • Pagination controls may simplify to 'Previous'/'Next' icons and a page number input.
