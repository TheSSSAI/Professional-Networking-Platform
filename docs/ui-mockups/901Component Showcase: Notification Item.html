Component Variations

Variant Name: "Unread Connection Request"

An incoming connection request that has not yet been seen by the user. It has a distinct background color and an 'unread' indicator dot.

Variant Name: "Unread Reaction (Batched)"

A notification for multiple similar events (e.g., likes) on a single piece of content, grouped together. Shown in the unread state.

Variant Name: "Unread Comment"

A notification for a new comment on the user's content, shown in the unread state.

Variant Name: "Read Notification"

A standard notification that has already been seen by the user. It has a neutral background and no 'unread' indicator.
