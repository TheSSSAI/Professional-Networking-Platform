Molecular Component: ReactionPicker

This component allows users to select a reaction from a set of options. It appears on hover/focus for desktop and can be triggered by long-press on touch devices.

Default State

This is the base state of the component, showing only the primary reaction button.

Expanded State (Hover/Focus)

Hover over or use the Tab key to focus the "Like" button to reveal more reactions.

Expanded State (Touch Simulation)

This is a static demonstration of how the component would look after a long-press on a touch device.
