FileUpload Component

A component for uploading files with a dropzone and progress states. Built using atomic components like Button, Spinner, and Icon.

Idle State

The default state, prompting the user to select a file or drag and drop.

Uploading State

Displays a progress bar and a spinner while the file is being uploaded.
