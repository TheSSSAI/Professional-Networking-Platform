Variant Name : "default"

The default DataTable implementation supports various data types in cells, including text, molecular components like UserBadge and Badge, and interactive elements like the ActionMenu. This variant is designed for desktop-first admin interfaces and becomes horizontally scrollable on smaller viewports.
