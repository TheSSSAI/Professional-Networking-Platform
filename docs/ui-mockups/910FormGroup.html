FormGroup Component

This molecular component groups a label, an input, and a message, providing structure and consistent styling for form fields.

Variants and States

The FormGroup adapts its styling based on its state. Below are demonstrations with a TextInput and TextArea.

Default State

The standard appearance for a form field, ready for user input. It includes helper text for additional context.

This will be displayed on your public profile.

Error State

Used to indicate that the user's input has failed validation. The .form-group--error class is applied, and ARIA attributes (aria-invalid, aria-describedby) are used for accessibility.

Please enter a valid email address.

Success State

Used to provide positive feedback after successful validation, such as checking for username availability. The .form-group--success class is applied.
