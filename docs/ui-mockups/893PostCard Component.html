Component Variants

Variant Name: "Default (Text-Only)"

The standard appearance for a post containing only text. This is the most basic form of the PostCard component.

Variant Name: "With Images"

A post containing text and one or more images. The image grid adapts to show 1, 2, 3, or 4 images in a visually appealing layout.

Variant Name: "With Link Preview"

A post containing text and a rich link preview card, which is automatically generated when a URL is detected. It includes an image, title, and description from the linked page.
