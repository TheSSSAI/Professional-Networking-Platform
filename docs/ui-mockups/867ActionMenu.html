ActionMenu

A molecular component for displaying a set of contextual actions in a dropdown menu, triggered by an icon button.

States

The menu appears when the container element gains focus from within, such as by tabbing to the button. A JavaScript implementation would typically handle clicks and closing the menu on outside clicks.

This state is shown here for demonstration using `data-state="open"`. In a real application, this would be triggered by focus or a click event.

The standard ActionMenu used for most contextual actions like 'Edit' and 'Delete'. It provides clear separation for destructive actions.
