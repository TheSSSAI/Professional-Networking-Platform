Confirmation Dialog

This component requires user confirmation for an action.

Variant: "Standard"

Used for non-destructive confirmations, like saving changes or proceeding with a standard workflow.

Variant: "Destructive"

Used to confirm irreversible actions, such as deleting data or removing a connection. The primary action button is styled to indicate danger.

Save Changes?

You have unsaved changes. Are you sure you want to save and continue?

Delete Account?

This action is permanent and cannot be undone. All your data, including your profile and posts, will be permanently erased.
