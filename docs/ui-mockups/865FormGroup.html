FormGroup Component

A molecular component that combines a label, text input, and error message to create a standard, accessible form field.

Default State

The standard appearance for a form field, ready for user input. The :focus state is also demonstrated here.

This is an error message.

Error State

This state is activated by adding the .form-group--error class to the parent container. This visually indicates an error and displays the associated error message. For accessibility, the input has aria-invalid="true" and is linked to the error message via aria-describedby.
