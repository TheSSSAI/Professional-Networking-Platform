Confirmation Modal Component

This component is used to confirm critical user actions. Below are interactive examples of its variants.

Confirm Action

Are you sure you want to proceed? This will update your profile information.

Delete Account?

This action is permanent and cannot be undone. All your data will be permanently erased.

Screen Variations

Variant Name: "Standard"

Used for confirming non-destructive but significant actions, like saving major changes.

Confirm Action

Are you sure you want to proceed? This will update your profile information.

Variant Name: "Destructive"

Used for confirming irreversible actions like deletion. The primary action button uses the 'danger' color to alert the user.
