Reaction Picker

A control for selecting one of several reactions to a piece of content.

Default (Hover/Focus to expand)

The picker reveals its options when the user hovers their mouse over it or focuses on it using a keyboard.

Selected State

When a user has already reacted, the trigger button reflects their selection. Hovering still allows them to change their reaction.

Touch Device Simulation

On touch devices, a long-press would trigger the expanded state. This is simulated here by adding a class.
