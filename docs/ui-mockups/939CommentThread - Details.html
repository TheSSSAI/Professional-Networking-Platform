Description

An organism component that displays a list of comments for a piece of content and includes an input form for adding new comments. It handles loading and loaded states.

Design System Alignment

This component is an organism composed of molecular components like UserBadge and form components like CommentInput. It heavily relies on the design system's tokens for spacing, typography, colors, and layout structure.
